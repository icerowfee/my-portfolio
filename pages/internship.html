Major Projects & Assigned Tasks

Create a Company Website

Duration: 2 months

The goal was to create a company website where clients could browse and view the sample services and products offered by D'sign Depot Central PH Inc. The website features dedicated pages for service and product showcases, as well as a contact page that allows users to request a quote directly online. As the backend and frontend developer, I was responsible for designing and implementing both the user interface and the server-side logic. On the frontend, I built responsive layouts using HTML, CSS (TailwindCSS), and Alpine.js to ensure a seamless experience across devices. On the backend, I developed the quote request functionality, which processes and stores client inquiries submitted through the contact form. This involved setting up form validation, handling submissions, and integrating email notifications for the company. The project resulted in a modern, user-friendly website that improved the company's online presence and streamlined the process for potential clients to request quotes and learn about our offerings.

Key Achievements:

  • • Improved my skills and knowledge in using Laravel for backend development
  • • Enhanced my skills in frontend and backend development using HTML, CSS (TailwindCSS), and Alpine.js
  • • Expanded my knowledge in web development best practices and user experience design
